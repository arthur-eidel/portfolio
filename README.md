# Portfolio

A couple of things I've built. Each project has its own folder and README.

## [tram7-analyzer](tram7-analyzer/)

Python data pipeline analysing the punctuality of Zurich's tram line 7, using
the open VBZ SOLL-IST dataset (~1.4M rows). Six-stage pipeline plus an
interactive Streamlit dashboard. pandas · Plotly · Streamlit.

## [physio-scraper](physio-scraper/)

Lead-generation scraper that finds physiotherapy practices across
German-speaking Switzerland, classifies each one by web presence, and exports
the prospects to Excel. OpenStreetMap + optional Google Places. requests ·
BeautifulSoup · openpyxl.
